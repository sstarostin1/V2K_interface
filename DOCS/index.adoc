== Обзор проекта

VCAS Server Viewer - графическое приложение для просмотра архитектуры сервера каналов VCAS в системе ВЭПП-2000. Подробное описание в xref:overview.adoc[Общем описании проекта].

== Архитектура системы

=== Общая архитектура
xref:architecture/system-design.adoc[Обзор компонентов и их взаимодействий].

=== Компоненты
* xref:components/MainWindow.adoc[MainWindow] - Главное окно приложения с двухпанельным интерфейсом.
* xref:components/VCASClient.adoc[VCASClient] - Клиент для работы с VCAS сервером.
* xref:components/MockVCASServer.adoc[MockVCASServer] - Тестовый сервер для отладки.
* xref:components/Config.adoc[Config] - Конфигурация приложения с сохранением настроек.
* xref:components/ChannelTreeWidget.adoc[ChannelTreeWidget] - Виджет дерева каналов с множественным выбором.
* xref:components/ChannelTableWidget.adoc[ChannelTableWidget] - Таблица деталей каналов и директорий.
* xref:components/NavigationHandler.adoc[NavigationHandler] - Обработчик навигации и клавиш.

=== Модели данных
* xref:components/ChannelData.adoc[ChannelData] - Модель данных канала с поддержкой временных рядов.

=== Система визуализации
* xref:components/PlotManager.adoc[PlotManager] - Центральный менеджер системы графиков.
* xref:components/PlotDockWindow.adoc[PlotDockWindow] - Независимое окно с графиками каналов.
* xref:components/PlotContainer.adoc[PlotContainer] - Контейнер для компоновки графиков.
* xref:components/DraggablePlotWidget.adoc[DraggablePlotWidget] - Интерактивный виджет графика с drag-and-drop.
