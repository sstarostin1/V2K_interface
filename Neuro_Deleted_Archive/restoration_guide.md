# Инструкция по восстановлению удаленных компонентов

## Общий процесс
1. Определите, какой файл нужно восстановить.
2. Скопируйте файл из соответствующей поддиректории в корень проекта.
3. Удалите запись о файле из README.md.
4. Проверьте функциональность приложения.

## Восстановление файлов
- debug_channels.py: Скопировать из deleted_files/, удалить из README.md, протестировать диагностику каналов.
- test_channels_simple.py: Скопировать из deleted_files/, удалить из README.md, протестировать структуру каналов.
- test_dependencies.py: Скопировать из deleted_files/, удалить из README.md, протестировать зависимости.
- info.docx: Скопировать из deleted_files/, удалить из README.md.
- VCAS descr.docx: Скопировать из deleted_files/, удалить из README.md.
- vcas_viewer.log: Скопировать из deleted_files/, удалить из README.md.

## Проверка после восстановления
- Запустите python main.py --mock-server для проверки приложения.
- Если тестовые файлы, запустите их отдельно.

## Контакты
Если проблемы с восстановлением, обратитесь к разработчикам.
